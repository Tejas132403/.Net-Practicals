# .NET Technologies Lab (01CE1523)

## About

This repository contains the practical assignments completed for the **.NET Technologies Lab**. These practicals provide hands-on experience in C#, Object-Oriented Programming (OOP), ASP.NET, MVC Architecture, ADO.NET, REST APIs, and full-stack web application development.

---

# Learning Objectives

By completing these practicals, I learned to:

- Understand the .NET Framework and C# programming language.
- Apply Object-Oriented Programming concepts such as Classes, Objects, Constructors, Encapsulation, Inheritance, Polymorphism, and Interfaces.
- Handle runtime errors using Exception Handling.
- Develop ASP.NET Web Forms applications using server controls and validation.
- Manage user sessions and cookies in web applications.
- Design web applications using the MVC (Model-View-Controller) architecture.
- Create responsive user interfaces using HTML Helpers and Layouts.
- Perform CRUD (Create, Read, Update, Delete) operations using ADO.NET.
- Connect applications with SQL Server databases.
- Generate reports using Crystal Reports.
- Build RESTful APIs using ASP.NET Core.
- Organize projects using proper folder structures and reusable components.
- Use Git and GitHub for version control and project management.
